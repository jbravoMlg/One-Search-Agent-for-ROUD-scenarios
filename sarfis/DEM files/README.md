# DEM availability

The Digital Elevation Model (DEM), alternate high-resolution surface/image data, and demonstration videos associated with the field experiment are not distributed through this Git repository because of their size and release constraints.

They are available from the corresponding author upon reasonable request. When supplied, place the DEM files in this directory while preserving the names expected by the GUI:

- `UMA_SEG_24_dsm_200cm.xyz` or its generated `UMA_SEG_24_dsm_200cm.mat` cache;
- optional alternate data named `UMA_SEG_24_dsm_5cm.xyz` or `UMA_SEG_24_dsm_5cm.mat`.

Do not add requested DEMs, generated `.mat` caches, or videos to this repository.
